# React Checkpoint

Store website using https://fakestoreapi.com/ for the backend.

## Technologies used:

- Typescript
- Next.js
- Redux
- Sass
- React Testing Library

## Installation

- Clone or download the repo.

- run `yarn` to install dependencies.

- run `yarn build`

- run `yarn start`

- go to http://localhost:3000

## Test

- run `yarn test` or `yarn test --coverage`
